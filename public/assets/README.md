Image upload guide:

- Category images live in `public/assets/categories`.
- Team images live in `public/assets/team`.
- To use your own images, replace the matching file path in `src/siteData.js`.
- Recommended real image size: 1200x900 JPG/WebP for category/gallery images.
- Recommended team image size: 800x1000 JPG/WebP.
- Keep filenames simple, for example `dj-vehicle.jpg`, then update `image: '/assets/categories/dj-vehicle.jpg'`.
